================================================================================
                            RESPONSIVIDADE
================================================================================

BREAKPOINTS:
─────────────────────────────────────────────────────────────────────────────

  Desktop: > 1100px
  └─ Layout: 2 colunas para seções principais
  └─ Grid: Até 5 colunas para categorias/desejos

  Tablet: 900px - 1100px
  └─ Layout: Ajuste de colunas
  └─ Grid: Até 3 colunas

  Mobile: < 900px
  └─ Layout: 1 coluna (full width)
  └─ Grid: 2 colunas para categorias/desejos
  └─ Padding: Reduzido para 12px
  └─ Font Size: Ligeiramente reduzido


================================================================================
                          NOTAS IMPORTANTES
================================================================================

1. CONSISTÊNCIA: Sempre use as cores e fontes definidas neste guia
2. ACESSIBILIDADE: Mantenha contraste adequado entre texto e fundo (mínimo 4.5:1)
3. RESPONSIVIDADE: Teste em todos os breakpoints
4. GRADIENTES: Use gradientes para adicionar profundidade visual
5. SOMBRAS: Use sombras para hierarquia e profundidade
6. ESPAÇAMENTO: Mantenha consistência no espaçamento
7. TRANSIÇÕES: Use transições para feedback visual suave
8. FORMATAÇÃO: Sempre formate valores monetários em BRL (R$ X.XXX,XX)
9. DATAS: Use formato brasileiro (DD/MM/YYYY)
10. ÍCONES: Use ícones com cores consistentes com o contexto
11. TEMA ESCURO: Sempre use cores de fundo escuro para reduzir fadiga visual
12. VERDE VIBRANTE: Use verde (#10b981) para destaques e ações positivas
